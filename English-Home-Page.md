XX-Net
========
Anti-censorship tools  
A Reborn Goagent  
A firewall circumvention toolkit  


Functions
==========
* A stable, ease-of-use and fast proxy based on GAE
* Friendly web UI

Support platform
================
* XP, need tcpip.sys patch, like tcp-z  
* Win7/8/10  
* Linux  
* Mac
